Loja de Roupas "Estilo Dev"

O objetivo é construir uma loja virtual completa, focada em moda para desenvolvedores.

Funcionalidades Principais:

  • Página inicial com produtos em destaque.
  • Página de catálogo com filtros (categoria, tamanho, cor).
  • Página de detalhes do produto.
  • Carrinho de compras funcional.
  • Formulário de checkout (simulado).

Tecnologias Utilizadas

Para construir este projeto, usaremos as seguintes tecnologias e ferramentas:

HTML5

Estrutura semântica e moderna para todo o conteúdo do site.

CSS3

Estilização avançada com Flexbox, Grid e variáveis para um design responsivo.

JavaScript (ES6+)

Interatividade no lado do cliente e lógica para o carrinho de compras.

Node.js / Express

Servidor de desenvolvimento e base para futuras APIs e integrações.
